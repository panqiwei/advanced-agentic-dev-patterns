<Band tone="paper-raised" dense>
  <h2 class="card-title">Causal Models</h2>
  <p class="card-subtitle">Causal Models — a formal tool encoding causal influence between variables</p>
</Band>

<Band tone="tan">
  <CoreDef>Structural equation models (SEM) use the asymmetric equation D ≔ C to encode causal direction — unlike the symmetric mathematical D = C. In this model, intervention = delete V's structural equation and treat V as exogenous.</CoreDef>
</Band>

<Band tone="paper-raised" grow>
  <div class="sem-components">
    <div class="sem-section">
      <Kicker>Core components</Kicker>
      <KV label="Endogenous vars">Values set by the model's own equations (appear on the left)</KV>
      <KV label="Exogenous vars">Values set outside the model (appear only on the right)</KV>
      <KV label="Structural equation">D ≔ C (asymmetric, direction-encoding)</KV>
      <KV label="Acyclicity">An exogenous assignment uniquely determines every endogenous variable</KV>
    </div>
    <div class="sem-section">
      <Kicker>Intervention steps</Kicker>
      <div class="op-steps">
        <span class="op-step">① Delete V's structural equation</span>
        <span class="op-step">② Treat V as exogenous, set to target value</span>
        <span class="op-step">③ Leave other equations untouched</span>
        <span class="op-step">④ Solve for the remaining variables</span>
      </div>
    </div>
  </div>
</Band>

<Band tone="paper-sunk" dense>
  <div class="accuracy-section">
    <Kicker>Accuracy standards (contested)</Kicker>
    <div class="accuracy-row">
      <span class="acc-name">Counterfactual sufficiency</span>
      <span class="acc-desc">The family of counterfactuals the model entails are all true</span>
    </div>
    <div class="accuracy-row">
      <span class="acc-name">Modularity</span>
      <span class="acc-desc">Each equation can be broken by an independent intervention — the precondition for intervention</span>
    </div>
    <div class="accuracy-row">
      <span class="acc-name">Stability</span>
      <span class="acc-desc">Adding more variables doesn't overturn existing causal verdicts</span>
    </div>
  </div>
</Band>
