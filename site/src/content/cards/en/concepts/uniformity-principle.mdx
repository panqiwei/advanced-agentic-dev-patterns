  <h2 class="card-title">Uniformity Principle</h2>
  <p class="card-subtitle">Uniformity Principle — "the future will resemble the past": the core presupposition of induction, unprovable</p>
</Band>

<Band tone="tan">
  <CoreDef>The uniformity principle (UP) is what inductive inference presupposes: "unobserved cases will resemble observed ones." Hume's dilemma: UP can be established neither by demonstrative reasoning (its negation yields no contradiction) nor by probabilistic reasoning (that would circularly presuppose UP itself). Conclusion: no argument supports UP — inductive inference has no rational foundation.</CoreDef>
</Band>

  <div class="ai-mapping">
    <Kicker>Mapping to AI engineering</Kicker>
    <KV label="Train distribution = test distribution">The core assumption of machine learning — this is UP in technical dress. When distributions match, models work.</KV>
    <KV label="Distribution shift">UP failures in the wild — production deviates from training distribution, model performance collapses. An expected UP failure mode.</KV>
    <KV label="Context rot">The reliability assumption over early context — assumes early-token semantics stay consistent across the whole context</KV>
    <KV label="Okasha's objection">Quantifier-shift fallacy: different inductive inferences rest on different empirical premises — there is no shared UP</KV>
