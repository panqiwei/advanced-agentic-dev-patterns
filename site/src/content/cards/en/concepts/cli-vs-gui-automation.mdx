<Band tone="paper-raised" dense>
  <h2 class="card-title">CLI vs GUI Automation</h2>
  <p class="card-subtitle">two foundational paths for driving existing software with agents</p>
</Band>

<Band tone="tan">
  <CoreDef>Not a tactical difference but a foundational philosophical split on how an agent should relate to software. GUI has the agent operate the UI like a human; CLI wraps software in a structured interface so the agent calls functionality directly.</CoreDef>
</Band>

<Band tone="paper-raised" grow>
  <div class="compare-table">
    <div class="ct-header">
      <span>dimension</span><span class="col-gui">GUI Automation</span><span class="col-cli">CLI Generation</span>
    </div>
    <div class="ct-row"><span>interface</span><span>pixels / accessibility tree</span><span>command strings / JSON</span></div>
    <div class="ct-row"><span>build-time cost</span><span class="cost-low">low (no internal analysis)</span><span class="cost-high">high (generate / test / maintain)</span></div>
    <div class="ct-row"><span>run-time cost</span><span class="cost-high">high (screenshot tokens + slow)</span><span class="cost-low">low (plain text, ms response)</span></div>
    <div class="ct-row"><span>stability</span><span class="cost-high">low (breaks on UI redesign)</span><span class="cost-low">high (CLI commands are stable)</span></div>
    <div class="ct-row"><span>parallelism</span><span class="cost-high">hard (screen is exclusive)</span><span class="cost-low">naturally concurrent</span></div>
    <div class="ct-row"><span>auditability</span><span class="cost-high">weak (screenshot trail)</span><span class="cost-low">strong (command history is the log)</span></div>
  </div>
</Band>

<Band tone="paper-sunk" dense>
  <div class="use-cases">
    <div class="uc-col">
      <span class="uc-label col-gui">GUI wins at</span>
      <span class="uc-text">fully closed-source software · visual-judgment tasks · one-off ad-hoc operations · human-in-the-loop flows</span>
    </div>
    <div class="uc-col">
      <span class="uc-label col-cli">CLI wins at</span>
      <span class="uc-text">repeatable structured workflows · software with a data model · high concurrency · long-running cross-session tasks</span>
    </div>
  </div>
</Band>
