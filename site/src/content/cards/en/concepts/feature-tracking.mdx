<Band tone="tan">
  <CoreDef>An externalized, machine-readable task list — a JSON feature list the agent can read for global progress and update completion against. It blocks both "one-shotting" (doing it all in one go) and "premature victory" (declaring done too early).</CoreDef>
</Band>
