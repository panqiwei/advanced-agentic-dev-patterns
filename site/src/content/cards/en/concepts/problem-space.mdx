<Band tone="tan">
  <CoreDef>A problem exists because we know the test (what the goal is) but cannot directly produce a structure that passes it. Four elements: initial state + goal state + move operators + state space. Key insight: the same problem has wildly different difficulty under different representations — representation change is meta-level search.</CoreDef>
</Band>
