<Band tone="paper-raised" dense>
  <h2 class="card-title">Implicit Loop Architecture</h2>
  <p class="card-subtitle">Implicit Loop — no predefined path, behavior indirectly constrained by tools + prompt + permissions + feedback</p>
</Band>

<Band tone="tan">
  <CoreDef>Agents run inside an open feedback loop: gather context → take action → verify work → repeat. Each step is decided by the model; there is no hard-coded "first do X". Isomorphic to an OS kernel: event-driven, not flow-driven.</CoreDef>
</Band>
