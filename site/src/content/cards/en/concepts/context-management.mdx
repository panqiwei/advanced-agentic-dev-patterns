<Band tone="paper-raised" dense>
  <h2 class="card-title">Context Management</h2>
  <p class="card-subtitle">Context Management — an LLM's working memory has to be programmed explicitly</p>
</Band>

<Band tone="tan">
  <CoreDef>LLMs suffer from anterograde amnesia — weights are frozen, the context window resets every session. Human colleagues accumulate context, consolidate memories, grow expertise; an LLM does none of that natively. Context is working memory, and you have to program it.</CoreDef>
</Band>

<Band tone="paper-raised" grow>
  <div class="management-layers">
    <div class="ml-section">
      <Kicker>within the context window</Kicker>
      <KV label="compaction">built into Claude Agent SDK: compress history as context nears its limit</KV>
      <KV label="tool-result handling">selectively keep or drop tool-call results</KV>
    </div>
    <div class="ml-section">
      <Kicker>across sessions (externalized state)</Kicker>
      <KV label="progress file">claude-progress.txt: write a structured progress summary at each session's end</KV>
      <KV label="git history">commit messages as an auditable change log</KV>
      <KV label="feature list">an objective completion signal that bridges the gap between sessions</KV>
    </div>
  </div>
</Band>
