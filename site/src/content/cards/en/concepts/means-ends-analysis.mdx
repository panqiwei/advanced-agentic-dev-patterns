  <p class="card-subtitle">Means-Ends Analysis — compare current state to goal, apply operators that reduce the difference</p>

<Band tone="tan">
  <CoreDef>The core heuristic of GPS (General Problem Solver): identify the difference between current and goal states, pick an operator that shrinks it, and recurse on unmet preconditions. Not blind search — difference-driven goal decomposition. Newell & Simon 1972.</CoreDef>
</Band>

<Band tone="paper-raised" grow>
  <div class="mea-cycle">
    <Kicker>The MEA three-step cycle</Kicker>
    <div class="step-row s1">
      <span class="step-n">①</span>
      <div>
        <span class="step-name">Compare</span>
        <span class="step-desc">current state S vs goal G → identify difference D</span>
      </div>
    </div>
    <div class="step-row s2">
      <span class="step-n">②</span>
      <div>
        <span class="step-name">Select</span>
        <span class="step-desc">find an operator O that eliminates D</span>
      </div>
    </div>
    <div class="step-row s3">
      <span class="step-n">③</span>
      <div>
        <span class="step-name">Recurse</span>
        <span class="step-desc">if O's preconditions aren't met → make meeting them a new subgoal → recurse MEA</span>
      </div>
    </div>
  </div>
  <div class="modern-map">
