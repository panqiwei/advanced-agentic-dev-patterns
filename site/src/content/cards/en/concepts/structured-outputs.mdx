  <p class="card-subtitle">Structured Outputs — force format compliance via JSON Schema + CFG, but at a hidden cost</p>

<Band tone="tan">
  <CoreDef>The implementation chain: JSON Schema → context-free grammar (CFG) → token mask (only locally legal tokens allowed). Effect is dramatic: OpenAI reports JSON mode format-error rate 11.97%, Structured Outputs drops to 0.1%. The hidden cost is "trajectory bias" — enforcing format reshapes the model's reasoning path and can reduce task accuracy. Format-correct does not mean answer-correct.</CoreDef>
</Band>

    <Kicker>Hidden costs & engineering principles</Kicker>
    <KV label="Trajectory bias">Forced format constrains the reasoning path — for some tasks, free text scores higher</KV>
    <KV label="Anthropic grammar cache">A fixed Schema triggers server-side caching; reusing one Schema reduces latency</KV>
    <KV label="Progressive structuring">Reason freely first, extract structure only in the last step — separate thinking from formatting</KV>
    <KV label="Use only when needed">Reach for Structured Outputs for tool calls / API wiring; human-read output needs no enforcement</KV>
