<Band tone="paper-raised" dense>
  <h2 class="card-title">Interventionist Theory</h2>
  <p class="card-subtitle">Interventionist Theory — X causally influences Y iff some intervention on X changes Y</p>
</Band>

<Band tone="tan">
  <CoreDef>Causation is not a static relation but a systematic answer to "what happens if you manipulate X?" Difference from Lewis's counterfactuals: counterfactuals are evaluated via nearest possible worlds; interventionism evaluates by actual manipulation after fixing causally irrelevant factors.</CoreDef>
</Band>

<Band tone="paper-raised" grow>
  <div class="woodward-defs">
    <Kicker>Woodward's definition chain (non-viciously circular)</Kicker>
    <div class="def-row">
      <span class="def-n">1</span>
      <div>
        <span class="def-title">Intervention variable I</span>
        <span class="def-body">I affects X; I makes X independent of other variables; every I→Y path goes through X; I is independent of non-X-mediated influences on Y</span>
      </div>
    </div>
    <div class="def-row">
      <span class="def-n">2</span>
      <div>
        <span class="def-title">Direct effect</span>
        <span class="def-body">There is an intervention on X that, holding other variables fixed, changes Y</span>
      </div>
    </div>
  </div>
  <div class="sem-tools">
    <Kicker>Structural-equation formalization</Kicker>
    <KV label="Asymmetric structural equations">D := C encodes the direction from C to D, unlike a symmetric correlation</KV>
    <KV label="Modularity">Each equation can be broken independently by intervention without affecting the others</KV>
    <KV label="A/B testing analog">Fix everything else, vary one variable — the engineering realization of intervention</KV>
  </div>
  <div class="two-paths">
    <span class="tp-label">Causation-first vs counterfactual-first</span>

  </div>
</Band>
