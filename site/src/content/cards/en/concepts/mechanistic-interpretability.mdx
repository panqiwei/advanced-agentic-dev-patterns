<Band tone="tan">
  <CoreDef>Reverse-engineer the neural net's actual computation — not "what the model did," but "how it did it." Core toolkit: features (monosemantic directions) → attribution graphs (causal chains of features) → circuits (cross-layer subgraphs). Anthropic 2024's dissection of Claude 3 Sonnet.</CoreDef>
</Band>

<Band tone="paper-raised" grow>
  <div class="tool-stack">
    <Kicker>Three-layer toolkit</Kicker>
    <div class="ts-row"><span class="ts-label">Features</span><span class="ts-desc">single semantic directions in activation space; superposition hypothesis: one neuron = many features linearly superposed (overcomplete basis)</span></div>
    <div class="ts-row"><span class="ts-label">Attribution Graphs</span><span class="ts-desc">trace backward which feature activations drove the output — causal chain, not correlation</span></div>
    <div class="ts-row"><span class="ts-label">CLT Cross-Layer Transcoder</span><span class="ts-desc">use sparse autoencoders to decompose MLPs into interpretable features; enables cross-layer circuit wiring</span></div>
  </div>
  <div class="findings">
    <Kicker>Key findings</Kicker>
    <div class="f-row halluc">
      <span class="f-tag">HALLUCINATION</span>
      <span class="f-text">confident-wrong features coexist with "insufficient knowledge" features — hallucination is a failure of feature competition, not randomness</span>
    </div>
    <div class="f-row cot">
      <span class="f-tag">UNFAITHFUL COT</span>
      <span class="f-text">the reasoning trace isn't always the real execution plan; the model may decide the answer first and rationalize after</span>
    </div>
    <div class="f-row jail">
      <span class="f-tag">JAILBREAK TENSION</span>
      <span class="f-text">safety features and capability features interfere; attacks can bypass specific safety directions without hitting other functions</span>
    </div>
  </div>
</Band>
