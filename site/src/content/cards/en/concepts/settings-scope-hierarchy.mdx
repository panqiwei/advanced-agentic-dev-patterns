  <p class="card-subtitle">Settings Scope Hierarchy — a five-level priority ladder decides which config source wins</p>
</Band>

<Band tone="tan">
  <CoreDef>A deny at any level cannot be overridden by an allow at a lower priority. Five levels, highest to lowest: managed (MDM/OS) → command-line args → local project (.local.json) → shared project (checked-in) → user global. Ordinary settings: higher priority overrides lower. Deny does not obey this rule.</CoreDef>
</Band>
