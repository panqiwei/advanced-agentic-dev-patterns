<Band tone="paper-raised" dense>
  <h2 class="card-title">Token vs Type Causation</h2>
  <p class="card-subtitle">Token vs Type Causation — the basic dichotomy of causal claims, crossed with constants/variables into a full 2×2 matrix</p>
</Band>

<Band tone="tan">
  <CoreDef>Token causation: "this drought caused this famine" — a relation between specific events. Type causation: "drowsy driving causes car crashes" — a relation between event types. Crossing the token/type split with constants/variables yields the full 2×2: token causation, type causation, token influence, type influence. Asymmetry is key: there can be token causation without corresponding type causation (a ball ricochets off a tree into a hole), and type causation without corresponding tokens (plutonium causes death — no one has ever drunk any).</CoreDef>
</Band>
