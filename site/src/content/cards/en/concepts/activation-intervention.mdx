<Band tone="tan">
  <CoreDef>Probes ask "is it encoded?"; interventions ask "does it causally drive?". Edit a mid-layer activation during the forward pass and watch the output shift — jumping from correlation to causal evidence.</CoreDef>
</Band>

<Band tone="paper-raised" grow>
  <div class="method-grid">

      <span class="method-name">Gradient-guided edit</span>
      <span class="method-tag">Li et al. 2022</span>
      <p>Gradient descent pushes the activation toward target state B'; can reach off-distribution states</p>
    </div>
    <div class="method-card m-linear">
      <span class="method-name">Linear vector addition</span>
      <span class="method-tag">Nanda et al. 2023</span>
      <p>x' ← x + α·v — one addition suffices; proves the representation itself is linear</p>
    </div>
    <div class="method-card m-patch">
      <span class="method-name">Activation patching</span>
      <span class="method-tag">Meng et al. 2022</span>
      <p>Clean run vs corrupted run — locates which layer stores a fact (ROME)</p>
    </div>
    <div class="method-card m-steer">
      <span class="method-name">Steering vectors</span>

    </div>
  </div>
