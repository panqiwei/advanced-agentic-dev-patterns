  <h2 class="card-title">Physical Symbol System Hypothesis</h2>
  <p class="card-subtitle">PSSH — a physical symbol system is necessary and sufficient for general intelligent action</p>
</Band>

<Band tone="tan">
  <CoreDef>Physical symbol system = a physical entity that manipulates symbol structures (creates/modifies/copies/destroys them). Two key properties: designation (expressions linked to objects) + interpretation (executing the designated process). PSSH: every general-intelligence system must be a physical symbol system, and being one suffices for intelligence.</CoreDef>
</Band>

<Band tone="paper-raised" grow>
  <div class="historical-chain">

    <div class="hc-row"><span class="hc-n">1</span><span class="hc-era">Formal logic (late 19c)</span><span class="hc-sig">Frege/Russell: symbols = meaningless tokens</span></div>
    <div class="hc-row"><span class="hc-n">2</span><span class="hc-era">Turing machine (1930s)</span><span class="hc-sig">machine-computability — laid the physical-realization foundation</span></div>
    <div class="hc-row"><span class="hc-n">3</span><span class="hc-era">Stored program (1940s)</span><span class="hc-sig">program as data: interpretation becomes physical reality</span></div>
    <div class="hc-row"><span class="hc-n">4</span><span class="hc-era">List processing (1956)</span><span class="hc-sig">introduces genuine designation — data structures can designate structures</span></div>
    <div class="hc-row"><span class="hc-n">5</span><span class="hc-era">LISP (1959)</span><span class="hc-sig">formalizes S-expressions, decoupling from specific machines</span></div>
  </div>
