<Band tone="paper-raised" dense>
  <h2 class="card-title">Meta-Harness</h2>
  <p class="card-subtitle">Meta-Harness — the standardized interface layer between an agent's brain and its execution environment</p>
</Band>

<Band tone="tan">
  <CoreDef>The Meta-Harness is the decoupling interface between the cognitive layer (Session) and the execution layer (Sandbox). OS-virtualization analogy: Session is the process, Harness is the syscall interface, Sandbox is the hardware resource pool. Decoupling lets the same agent logic run across different execution environments.</CoreDef>
</Band>

<Band tone="paper-raised" grow>
  <div class="three-layers">
    <Kicker>Three-layer interface stack</Kicker>
    <div class="layer l-session">
      <span class="l-name">Session layer</span>
      <span class="l-role">Agent cognition: goals, planning, tool selection</span>
      <span class="l-analogy">OS analogy → user process</span>
    </div>
    <div class="l-arrow">↕ Harness interface</div>
    <div class="layer l-harness">
      <span class="l-name">Harness layer</span>
      <span class="l-role">Tool routing, permission checks, state persistence</span>
      <span class="l-analogy">OS analogy → syscall interface</span>
    </div>
    <div class="l-arrow">↕ Sandbox interface</div>
    <div class="layer l-sandbox">
      <span class="l-name">Sandbox layer</span>
      <span class="l-role">Actual execution: filesystem, network, processes</span>
      <span class="l-analogy">OS analogy → hardware / kernel resources</span>
    </div>
  </div>
  <div class="design-rationale">
    <Kicker>Decoupling benefits</Kicker>
    <KV label="Portability">Same Session logic can switch between local / remote / container sandboxes</KV>
    <KV label="Security boundary">Harness is the permission enforcement point — Session cannot bypass</KV>
    <KV label="Observability">All tool calls pass through the Harness, yielding audit logs by construction</KV>
    <KV label="Long-running tasks">Cross-session state lives in Harness/Sandbox, not the context window</KV>
  </div>
</Band>
