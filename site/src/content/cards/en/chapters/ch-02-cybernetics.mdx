<Band tone="paper-raised">

  <p style="margin: 4px 0 0; font-size: 0.95rem; color: var(--ink-soft);">The model is the rudder; the harness is the helmsman. The one setting the course is the helmsman, not the rudder — cybernetics has called this <em>κυβερνήτης</em> since 1948.</p>
</Band>

<Band tone="paper-sunk">
  <Kicker>② FEEDBACK LAYERS</Kicker>
  <p style="margin: 4px 0 0; font-size: 0.95rem; color: var(--ink-soft);">Observe → error → correct. Negative feedback converges to homeostasis; positive feedback amplifies deviation. Four nested layers: token · turn · session · alignment.</p>
</Band>

<Band tone="paper-raised">
  <Kicker>③ REQUISITE VARIETY (ASHBY)</Kicker>
  <p style="margin: 4px 0 0; font-size: 0.95rem; color: var(--ink-soft);">Only variety can absorb variety. The harness must span the behaviors the model can produce — anything uncovered is a blind spot.</p>
</Band>

<Band tone="tan" grow>
  <Kicker>④ SECOND ORDER (VON FOERSTER)</Kicker>
  <p style="margin: 4px 0 0; font-size: 0.95rem; color: var(--ink-soft);">When you tune a prompt, you're inside, not outside. The observer is part of the system — when disturbance exceeds the first-order loop's correction range, let the first-order goal itself become a variable.</p>
</Band>
