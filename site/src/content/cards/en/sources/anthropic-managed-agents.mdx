<Band tone="tan">
  <CoreDef>An OS virtualizes hardware to offer universal abstractions to "programs not yet conceived." Managed Agents virtualize agent components (session / harness / sandbox) to offer universal interfaces to future harnesses. The core idea: a "meta-harness" — provide the interface, do not prescribe the implementation.</CoreDef>

  <div class="col">
    <div class="col__k">Pet → Cattle</div>
    <div class="col__v">A single container is an "irreplaceable pet." Decoupling each component turns it into independently swappable "cattle." Brain startup no longer waits on container provisioning: <span class="hl">p50 TTFT ↓60% · p95 ↓90%</span>.</div>
  </div>
  <div class="col">
    <div class="col__k">Structural Security Boundary</div>
    <div class="col__v">Credentials never enter the sandbox. Git tokens are injected into the remote; OAuth tokens live in a vault behind a dedicated proxy. Prompt injection cannot talk its way into what isn't there.</div>
  </div>
