<Band tone="paper-raised" dense>
  <h2 class="card-title">Context Rot</h2>
  <p class="card-subtitle">Model performance measurably degrades as input length grows — even on trivially simple tasks</p>
</Band>

<Band tone="tan" dense>
  <CoreDef>Methodological contribution — <span class="hl">hold task difficulty constant, vary only input length</span>, isolating input length itself as the degradation variable. All 18 frontier models, no exceptions: more tokens = degraded performance.</CoreDef>
</Band>

<Band tone="paper-raised" grow>
  <div class="rot-head">Three degradation mechanisms</div>
  <div class="mech">
    <div class="mch mch--1">
      <div class="mch-n">I</div>
      <div class="mch-body">
        <div class="mch-k">Needle-Question Similarity</div>
        <div class="mch-v">Low-similarity pairs succeed at short context but degrade at long context — degradation stems not from task difficulty but from input length itself</div>
      </div>
    </div>
    <div class="mch mch--2">
      <div class="mch-n">II</div>
      <div class="mch-body">
        <div class="mch-k">Distractor Interference</div>
        <div class="mch-v">A single distractor drops performance, four distractors drop it further — <span class="em">distractors hurt unevenly</span>, and the effect amplifies with input length</div>
      </div>
    </div>
    <div class="mch mch--3">
      <div class="mch-n">III</div>
      <div class="mch-body">
        <div class="mch-k">Haystack Structure Effect</div>
        <div class="mch-v">Counterintuitive — <span class="em">a coherently structured haystack hurts performance more than a randomly shuffled one</span>; attention is sensitive to input structure</div>
      </div>
    </div>
  </div>

  <div class="behavior">
    <Kicker>Model behavior differences · diverging strategies under the same pressure</Kicker>
    <div class="bv-row">
      <span class="bv-k">Claude</span>
      <span class="bv-v">Conservative — prefers abstention ("can't find the answer") when uncertain; lowest hallucination rate</span>
    </div>
    <div class="bv-row">
      <span class="bv-k">GPT</span>
      <span class="bv-v">Confident — facing a distractor, tends to generate a plausible but wrong answer; highest hallucination rate</span>
    </div>
  </div>
</Band>

<Band tone="paper-sunk" dense>
  <div class="takeaway">
    <span class="tk-k">Key takeaway</span>
    <span class="tk-v">NIAH benchmarks test only lexical retrieval — a high score doesn't guarantee reliable long-context capability. <span class="em">How information is presented</span> matters as much as whether it's in context at all.</span>
  </div>
</Band>
