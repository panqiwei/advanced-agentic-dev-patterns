  <CoreDef>Progressive disclosure — a skill can package an arbitrary amount of context in theory, because the agent has a file system and can execute code, so it never needs to read everything at once. Some operations (sorting, PDF parsing) are cheaper as code than as tokens — code is both tool and documentation.</CoreDef>
