  <p class="card-subtitle-on-ink">Three patterns for building apps that keep up with Claude</p>
</Band>

<Band tone="paper-raised">
  <div class="pat">
    <div class="pat__num">I</div>
    <div class="pat__body">
      <div class="pat__label">USE what Claude knows</div>
      <div class="pat__head">Reach for the general tools Claude already speaks</div>
      <div class="pat__desc">bash + text editor beats bespoke tools for every scenario. Claude Code's SWE-bench trajectory proves it: on the same toolset, capability keeps climbing. Skills, programmatic tool calling, and the memory tool are all compositions of bash + text editor.</div>
    </div>
  </div>
</Band>

<Band tone="tan" grow>
  <div class="pat">
    <div class="pat__num pat__num--hero">II</div>
    <div class="pat__body">
      <div class="pat__label">ASK "what can I stop doing?"</div>
      <div class="pat__head">Actively strip away outdated assumptions</div>
      <div class="pat__desc">Let Claude orchestrate its own actions (code execution ≫ piping results back through context) — BrowseComp 45.3% → 61.6%; let Claude manage its own context (skills + context editing); let Claude persist its own context (compaction + memory folder). Opus 4.6 compaction hits 84%, Sonnet 4.5 only 43%.</div>
    </div>
  </div>
</Band>

<Band tone="paper-sunk">
  <div class="pat">
    <div class="pat__num">III</div>
    <div class="pat__body">
      <div class="pat__label">SET boundaries carefully</div>
      <div class="pat__head">Set guardrails with care</div>
      <div class="pat__desc">Cache-aware (static first, don't swap models, mutate tools sparingly) · declarative tools (promote irreversible ops to typed tools) · continuous reevaluation (auto-mode uses a second Claude to judge bash safety — fewer bespoke tools needed).</div>
    </div>
  </div>
</Band>

<Band tone="ink" dense>
  <div class="punchline">
    <div class="pl__quote">"Structure or boundaries in an app should be pruned based on <span class="pl__em">'what can I stop doing?'</span> — because they may become the bottleneck on Claude's performance."</div>
    <div class="pl__attr">— the Bitter Lesson, agent edition</div>
  </div>
</Band>
