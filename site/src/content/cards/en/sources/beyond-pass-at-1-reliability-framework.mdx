<Band tone="tan" dense>
  <CoreDef>Reliability ≠ capability. A four-metric framework — <span class="bk">RDC</span> decay curve · <span class="bk">VAF</span> variance amplification factor · <span class="bk">GDS</span> graceful degradation · <span class="bk">MOP</span> meltdown onset point — each captures a dimension that pass@1 throws away.</CoreDef>
</Band>

<Band tone="paper-raised" grow>
  <div class="metrics">
    <div class="m m--1">
      <div class="m-head">
        <span class="m-abbr">RDC</span>
        <span class="m-full">Reliability Decay Curve</span>
      </div>
      <div class="m-meaning">pass^k's decay against task length — flatter is better</div>
      <div class="m-sig">Linear-regression slope RDS serves as the scalar summary</div>
    </div>
    <div class="m m--2">
      <div class="m-head">
        <span class="m-abbr">VAF</span>
        <span class="m-full">Variance Amplification Factor</span>
      </div>
      <div class="m-meaning">Ratio of long/short-task pass@1 variance — <span class="hl">high VAF ≥ 2.37 is a capability signal</span></div>
      <div class="m-sig">Counterintuitive: stable failure ≠ good</div>
    </div>
    <div class="m m--3">
      <div class="m-head">
        <span class="m-abbr">GDS</span>
        <span class="m-full">Graceful Degradation Score</span>
      </div>
      <div class="m-meaning">Weighted score for partial completion (0–1) — still discriminates models when pass@1 → 0</div>
      <div class="m-sig">Binary evaluation discards more and more information</div>
    </div>
    <div class="m m--4">
      <div class="m-head">
        <span class="m-abbr">MOP</span>
        <span class="m-full">Meltdown Onset Point</span>
      </div>
      <div class="m-meaning">Tool-call distribution entropy spikes suddenly — the agent slips into disordered behavior</div>
      <div class="m-sig">Paradox: frontier models have the highest meltdown rates</div>
    </div>
  </div>
</Band>
