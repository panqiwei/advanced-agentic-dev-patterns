<Band tone="tan">
  <CoreDef>Central thesis: "Agents Are Not Just a Model Problem. They Are an Execution Problem." Most agent failures aren't because the model is too weak — they're because the execution infrastructure doesn't fit. Classical OS abstractions (process, thread, file) were never designed for dynamic, semantically rich agent workloads. The OS itself needs to become agentic.</CoreDef>
</Band>

    <div class="pp-row pp-exec">

    <KV label="eBPF observability">Live tracing of agent call chains — non-invasive monitoring without modifying agent code</KV>
