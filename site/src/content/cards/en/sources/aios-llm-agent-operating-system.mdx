<Band tone="tan">
  <CoreDef>When multiple agents share a single LLM, the problems — scheduling, isolation, preemption/resume, memory — are structurally the same ones 1960s OSes faced around the CPU. AIOS maps all six classical OS modules onto LLM agent management; it's the most complete engineering instantiation of Karpathy's "LLM = CPU" analogy.</CoreDef>
</Band>
