  <h2 class="card-title">Circuit Tracing: Revealing Computational Graphs in Language Models</h2>
  <p class="card-subtitle">Anthropic Transformer Circuits Team (2025) — a full MI workflow from feature discovery to circuit tracing</p>
</Band>

<Band tone="tan">
  <CoreDef>Build an "interpretable replacement model": swap MLP layers with a cross-layer transcoder (CLT) and trace linear causal chains between sparsely activated features to produce an "attribution graph." Feature-to-feature interactions in the forward pass are linear — CLT bridges the MLP, attention patterns are frozen. The largest CLT matches the underlying model's top-1 token on 50% of prompts.</CoreDef>
</Band>

<Band tone="paper-raised" grow>
  <div class="workflow">
    <Kicker>Five-step workflow</Kicker>
    <div class="wf-row wf-1">
      <span class="wr-n">1</span>
      <span class="wr-desc">SAE/CLT feature extraction — read from the residual stream, write into downstream MLP outputs</span>
    </div>
    <div class="wf-row wf-2">
      <span class="wr-n">2</span>
      <span class="wr-desc">Build a local replacement model — freeze attention patterns, add an error-correction term, exactly reproduce the original model's output</span>
    </div>
    <div class="wf-row wf-3">
      <span class="wr-n">3</span>
      <span class="wr-desc">Generate attribution graph — nodes = active features / tokens / errors / logits; edges = linear effects</span>
    </div>
    <div class="wf-row wf-4">
      <span class="wr-n">4</span>
      <span class="wr-desc">Prune — keep the largest-contribution subgraph; typical 10x node compression at a 20% loss in explanatory power</span>
    </div>
    <div class="wf-row wf-5">
      <span class="wr-n">5</span>
      <span class="wr-desc">Perturbation validation — suppress/inject features, observe downstream effects, confirm causal chains</span>
    </div>
  </div>
  <div class="limits">
    <Kicker>Method limits</Kicker>
    <KV label="Missing QK-circuits">Captures OV-circuit information flow, but does not explain "why attend to this position"</KV>
    <KV label="~25% graph hit-rate">Only ~25% of prompts yield satisfying insight — good for exploratory research, not batch analysis</KV>
    <KV label="Suppression is hard to read">When a feature's role is to "prevent" an output, attribution graphs become harder to interpret</KV>
  </div>
</Band>
