    <div class="note">
      <span class="note__k">Grammar scope</span>
      <span class="note__v">Constrains only Claude's direct output; does not affect tool-call arguments, tool results, or thinking. Grammar state resets between sections — thinking is free, response is constrained.</span>
    </div>
    <div class="note note--warn">
      <span class="note__k">PHI / HIPAA warning</span>
      <span class="note__v">Schemas are cached separately, at a different privacy level. Do not put PHI in property names / enum / const / pattern regex — the schema is not message content, it's a system-layer artifact.</span>
    </div>
