  <p class="card-subtitle">A narrative tour of the Circuit Tracing + Biology of LLM papers — the internal mechanics behind ten key behaviors</p>

<Band tone="tan" dense>
  <CoreDef>Core metaphor: build a "microscope" for AI, in the spirit of how neuroscience studies the brain. Apply Circuit Tracing to Claude 3.5 Haiku and directly watch the computational paths the model walks for a given task.</CoreDef>
</Band>

<Band tone="paper-raised" grow>
  <div class="mosaic">
    <div class="find find--1">
      <div class="find__num">01</div>
      <div class="find__title">Shared multilingual representation</div>
      <div class="find__desc">Concept features are shared across languages — the model computes in an abstract "language of thought" space first, then translates to a concrete output language</div>
    </div>
    <div class="find find--2">
      <div class="find__num">02</div>
      <div class="find__title">Forward planning</div>
      <div class="find__desc">Writing a rhyming poem, the model preselects the rhyme word before starting the line, then "writes toward the target" — overturning the token-by-token improv hypothesis</div>
    </div>
    <div class="find find--3">
      <div class="find__num">03</div>
      <div class="find__title">Reasoning faithfulness</div>
      <div class="find__desc">Sometimes the model constructs reasoning backward from the answer (motivated reasoning) — interpretability tools catch it in the act</div>
    </div>
    <div class="find find--4">
      <div class="find__num">04</div>
      <div class="find__title">Hallucination mechanism</div>
      <div class="find__desc">Default behavior is refusal; a "known entity" feature suppresses it — spurious feature activation → hallucination</div>
    </div>
    <div class="find find--5">
      <div class="find__num">05</div>
      <div class="find__title">Jailbreak analysis</div>
      <div class="find__desc">Tension between syntactic-coherence features and safety mechanisms — the model leans toward finishing a started grammatical structure even after detecting danger</div>
    </div>
    <div class="find find--6">
      <div class="find__num">06</div>
      <div class="find__title">Mental arithmetic strategy</div>
      <div class="find__desc">The model invents its own parallel computation (approximate magnitude + exact last digit) — different from the textbook algorithm it claims to use in its "explanation"</div>
    </div>
  </div>
</Band>

<Band tone="paper-sunk" dense>
  <div class="limits">
    <span class="lim-k">Limits</span>
    <span class="lim-v">Attribution graphs capture only a portion of the computation · hours of human analysis per prompt · the replacement model may introduce mechanisms not present in the underlying one · future work needs to scale and extend to long CoT</span>
  </div>
</Band>
