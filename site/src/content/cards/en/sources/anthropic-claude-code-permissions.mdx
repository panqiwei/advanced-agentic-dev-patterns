<Band tone="tan">
  <CoreDef>Three tool tiers × three rule types × six modes × five-level scope form Claude Code's full permission model. Core semantics: deny-first — evaluation runs deny → ask → allow, the first match wins, deny always takes priority. A deny at any layer cannot be overridden by an allow elsewhere.</CoreDef>
</Band>
