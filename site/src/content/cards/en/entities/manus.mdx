<Band tone="tan">
  <CoreDef>Manus went through four architectural rebuilds and distilled a five-dimensional context engineering framework. Core economic insight: agent loops run prefill/decode at roughly 100:1, making prefix cache hit rate the primary production cost lever (Claude Sonnet: $0.30/MTok hit vs. $3.00/MTok miss — a 10x cost gap).</CoreDef>
</Band>

<Band tone="paper-raised" grow>
  <div class="five-dims">
    <Kicker>5-D Context Engineering Framework</Kicker>
    <div class="fd-row fd-offload">
      <span class="fd-name">Offloading</span>
      <span class="fd-desc">File system as unbounded external memory, zero token cost — move content "out of" context</span>
    </div>
    <div class="fd-row fd-reduce">
      <span class="fd-name">Reduction</span>
      <span class="fd-desc">Only recoverable compression (keep path/URL pointers) — prevent permanent information loss</span>
    </div>
    <div class="fd-row fd-retrieve">
      <span class="fd-name">Retrieval</span>
      <span class="fd-desc">File search tools as a structured retrieval layer — load on demand rather than all at once</span>
    </div>
    <div class="fd-row fd-isolate">
      <span class="fd-name">Isolation</span>
      <span class="fd-desc">Sub-agents run in separate context windows — isolation avoids cross-interference</span>
    </div>
    <div class="fd-row fd-cache">
      <span class="fd-name">Caching</span>
      <span class="fd-desc">Stable prefix + session affinity routing — maximize prefix cache hit rate</span>
    </div>
  </div>
  <div class="tech-decisions">
    <Kicker>Representative Technical Decisions</Kicker>
    <KV label="Logit Masking over Dynamic Tool Removal">Mutating the tools array breaks KV cache; use decode-time logit masking to constrain the action space instead</KV>
    <KV label="todo.md Attention Engineering">Update todo.md every iteration to keep the global plan near the recent end of context — counteracts "lost in the middle"</KV>
    <KV label="Error Retention Policy">Keep failure records so the model gets implicit belief updates — error-recovery behavior emerges naturally</KV>
  </div>
