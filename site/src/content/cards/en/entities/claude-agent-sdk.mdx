<Band tone="paper-raised" dense>
  <h2 class="card-title">Claude Agent SDK</h2>
  <p class="card-subtitle">Anthropic's agent development framework — formerly Claude Code SDK, lets agents use computers to get work done</p>
</Band>

<Band tone="tan">
  <CoreDef>Design philosophy: "give the agent a computer". The SDK lets agents use the file system, terminal, and tools — the same way humans use a computer. It runs on an implicit-loop architecture: gather context → take action → verify work → repeat. Behavior is shaped by tools, prompts, permissions, and feedback rather than a predefined graph.</CoreDef>
</Band>

    <Kicker>Long-running capability and limits</Kicker>
    <KV label="Built-in compaction">When context nears the limit, history is automatically compacted so work continues — a built-in SDK capability</KV>
    <KV label="Compaction alone is not enough">Anthropic's practice: externalized progress tracking is also needed (progress file + git history + feature tracking)</KV>
    <KV label="Two-agent architecture">An Initializer-Coder two-agent setup is what makes reliable cross-session work possible</KV>
    <KV label="With MCP">Claude Agent SDK + MCP tool invocation is the recommended agent-development combination</KV>
