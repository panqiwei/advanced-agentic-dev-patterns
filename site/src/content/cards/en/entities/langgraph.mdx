<Band tone="paper-raised" dense>
  <h2 class="card-title">LangGraph</h2>
  <p class="card-subtitle">LangChain's low-level agent-orchestration framework — defines a directed graph of nodes and edges via StateGraph</p>
</Band>

<Band tone="tan">
  <CoreDef>LangGraph is the main contrast paradigm to the implicit-loop architecture (Claude Agent SDK, Codex), representing one of the two poles of agent orchestration. Explicit-graph architecture: the flow is fixed at compile time, StateGraph defines nodes and edges of a directed graph, giving high predictability and easy debugging. Adopted by enterprises including Klarna, Uber, and J.P. Morgan, with durable execution, human-in-the-loop, and memory management as headline features.</CoreDef>
</Band>

<Band tone="paper-raised" grow>
  <div class="paradigm-compare">
    <Kicker>Two orchestration paradigms compared</Kicker>
    <div class="pc-grid">
      <div class="pc-cell pc-explicit">
        <span class="pc-name">LangGraph (explicit graph)</span>
        <span class="pc-when">Flow fixed at compile time</span>
        <span class="pc-pros">High predictability, easy to debug</span>
        <span class="pc-cons">Lower flexibility, relies on pre-set paths</span>
      </div>
      <div class="pc-cell pc-implicit">
        <span class="pc-name">Implicit loop (Claude/Codex)</span>
        <span class="pc-when">Model decides autonomously at runtime</span>
        <span class="pc-pros">High flexibility, no pre-set paths needed</span>
        <span class="pc-cons">Lower predictability, relies on model capability</span>
      </div>
    </div>
  </div>
  <div class="features">
    <Kicker>LangGraph signature features</Kicker>
    <KV label="Durable Execution">Execution state is persistable — tasks can resume from the breakpoint after interruption, fitting long-running workloads</KV>
    <KV label="Human-in-the-loop">Built-in support for human intervention — conditional nodes can pause and wait for a human decision</KV>
    <KV label="Enterprise adoption">Klarna / Uber / J.P. Morgan — the auditability of explicit graphs suits enterprise compliance needs</KV>
  </div>
</Band>
