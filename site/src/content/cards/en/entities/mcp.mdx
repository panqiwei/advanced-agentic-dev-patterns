<Band tone="paper-raised" dense>
  <h2 class="card-title">MCP — Model Context Protocol</h2>
  <p class="card-subtitle">Open protocol from Anthropic — standardizes how LLMs integrate with external tools and data sources</p>
</Band>

<Band tone="tan">
  <CoreDef>MCP uses JSON-RPC to specify call semantics between agent and tools — the standard path for an "augmented LLM" to plug into third-party tool ecosystems. Complementary to A2A: MCP handles Agent ↔ Tool, A2A handles Agent ↔ Agent.</CoreDef>
</Band>

  <div class="routes">
    <Kicker>Protocol Route vs. CLI Route</Kicker>
    <KV label="MCP Protocol Route">JSON-RPC cross-boundary communication, permission auditing, bidirectional streams — the "protocol layer" standardizing tool access</KV>
    <KV label="CLI-Anything Route">Auto-generated CLI + POSIX primitives — no protocol layer, explicitly avoids MCP</KV>
    <KV label="Layered Coexistence">The two routes are not zero-sum — a CLI implementation can be wrapped inside an MCP server and collaborate in layers</KV>
    <KV label="Mobile-MCP">Extends MCP from desktop/cloud to native mobile app capabilities via Android IPC</KV>
  </div>
