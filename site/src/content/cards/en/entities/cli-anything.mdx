  <h2 class="card-title">CLI-Anything</h2>
  <p class="card-subtitle">Auto-generates structured CLIs that turn any existing software into an agent-native callable tool</p>
</Band>

<Band tone="tan">
  <CoreDef>"Today's Software Serves Humans. Tomorrow's Users will be Agents." Released by the HKUDS team in March 2026; as of April 2026 it has 29,804 GitHub stars, covers 20+ mainstream software packages, and passes all 2,130 tests. One command triggers a fully automated 7-stage pipeline (Analyze → Design → Implement → Plan Tests → Write Tests → Document → Publish); outputs live under software/agent-harness/.</CoreDef>
</Band>

<Band tone="paper-raised" grow>
  <div class="design-principles">
    <Kicker>Four core design principles</Kicker>
    <div class="dp-row dp-auth">
      <span class="dp-name">Authentic Integration</span>
      <span class="dp-desc">Emit legitimate project files — rendering is delegated to the real software. Not a replacement, just a structured interface</span>
    </div>
    <div class="dp-row dp-flex">
      <span class="dp-name">Dual mode</span>
      <span class="dp-desc">No-arg enters a REPL (multi-step exploration); with args it runs as scripted — serves humans and agents simultaneously</span>
    </div>
    <div class="dp-row dp-ux">
      <span class="dp-name">Consistent UX</span>
      <span class="dp-desc">ReplSkin enforces a single visual language across all CLIs — isomorphic output lets agents transfer experience</span>
    </div>
    <div class="dp-row dp-native">
      <span class="dp-name">Agent-Native</span>
      <span class="dp-desc">Built-in --json, self-documenting --help, discoverable via which — POSIX primitives only, no extra SDK required</span>
    </div>
  </div>
  <div class="ecosystem">
    <Kicker>Ecosystem impact</Kicker>
    <KV label="CLI-Hub registry">Agents can browse and pip install any CLI autonomously — zero-human-in-the-loop agent-native software distribution</KV>
    <KV label="Cross-platform plugins">Official or community support on 8 platforms including Claude Code, Codex, GitHub Copilot CLI, and Goose</KV>
  </div>
</Band>
