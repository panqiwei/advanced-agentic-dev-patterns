<Band tone="tan">
  <CoreDef>Claude Code is not just a chat interface but a complete agent system that reads and writes files, executes Bash commands, calls MCP tools, and spawns sub-agents. Its permission system is a reference implementation of production-grade agent permission management: three-tier tool approval + deny-first rule semantics + five-level scope hierarchy + six permission modes, which combine with OS-level sandboxing for two-layer defense in depth.</CoreDef>
</Band>
