    <div class="pat__body">
      <div class="pat__label">USE what Claude knows</div>
      <div class="pat__head">用 Claude 已熟悉的通用工具</div>
      <div class="pat__desc">bash + text editor 胜过为每个场景定制专用工具。Claude Code 的 SWE-bench 进化证明：同一工具集上能力持续提升。Skills、programmatic tool calling、memory tool 都是 bash + text editor 的组合。</div>
    </div>
  </div>
</Band>

<Band tone="tan" grow>
  <div class="pat">
    <div class="pat__num pat__num--hero">二</div>
    <div class="pat__body">
      <div class="pat__label">ASK "what can I stop doing?"</div>
      <div class="pat__head">主动剥离过时假设</div>
      <div class="pat__desc">让 Claude 编排自身操作（代码执行 ≫ 回流 context）——BrowseComp 45.3% → 61.6%；让 Claude 管理自身 context（skills + context editing）；让 Claude 持久化自身 context（compaction + memory folder）。Opus 4.6 compaction 达 84%，Sonnet 4.5 仅 43%。</div>
    </div>
  </div>
</Band>

<Band tone="paper-sunk">
  <div class="pat">
    <div class="pat__num">三</div>
    <div class="pat__body">
      <div class="pat__label">SET boundaries carefully</div>
      <div class="pat__head">小心设置边界</div>
      <div class="pat__desc">Cache 优化（静态优先、不换模型、慎改工具）· 声明式工具（不可逆操作提升为 typed 工具）· 持续重评估（auto-mode 用第二个 Claude 判断 bash 安全性，可减少专用工具）</div>
    </div>
  </div>
</Band>
