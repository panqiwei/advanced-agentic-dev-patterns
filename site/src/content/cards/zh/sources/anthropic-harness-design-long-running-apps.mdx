  <h2 class="card-title">长时应用开发的 Harness 设计</h2>
  <p class="card-subtitle">GAN 启发的 Generator-Evaluator 架构——把生成与评估分离为独立 agent，解决「自评偏乐观」</p>
</Band>

<Band tone="tan">
  <CoreDef>三 agent 架构：<span class="hl">Planner</span>（1-4 句 prompt → 200+ feature spec）、<span class="hl">Generator</span>（逐 sprint 实现 + 自评）、<span class="hl">Evaluator</span>（Playwright MCP 实际操作应用验证）。核心洞察：agent 评估自己天然偏乐观，分离评估者后更容易调校为严格的批评者。</CoreDef>
</Band>
