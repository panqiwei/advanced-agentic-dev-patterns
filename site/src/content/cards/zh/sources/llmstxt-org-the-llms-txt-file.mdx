  <CoreDef>问题——LLM 的 context window 装不下整站内容，HTML 对 LLM 信噪比低。解法——在根路径放 <span class="mono hl">/llms.txt</span>，用 Markdown 策展一组 LLM 可读的"入口链接"。</CoreDef>
</Band>
