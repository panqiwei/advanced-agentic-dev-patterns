<Band tone="tan">

</Band>

<Band tone="paper-raised" grow>
  <div class="design-principles">
    <Kicker>四条核心设计原则</Kicker>
    <div class="dp-row dp-auth">
      <span class="dp-name">Authentic Integration</span>
      <span class="dp-desc">生成合法项目文件，渲染交给真实软件——不做替代品，只做结构化接口</span>
    </div>
    <div class="dp-row dp-flex">
      <span class="dp-name">双模式</span>
      <span class="dp-desc">无参进 REPL（多步试错），有参做脚本调用——同时服务人类和 agent</span>
    </div>
    <div class="dp-row dp-ux">
      <span class="dp-name">一致体验</span>
      <span class="dp-desc">ReplSkin 强制所有 CLI 用同一套视觉语言——同构产出让 agent 可迁移经验</span>
    </div>
    <div class="dp-row dp-native">
      <span class="dp-name">Agent-Native</span>
      <span class="dp-desc">--json 内建，--help 自文档，which 可发现——只用 POSIX 原语，无需额外 SDK</span>
    </div>
  </div>

    <KV label="跨平台插件">Claude Code / Codex / GitHub Copilot CLI / Goose 等 8 个平台有官方/社区支持</KV>
  </div>
