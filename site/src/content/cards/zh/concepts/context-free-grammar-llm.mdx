  <p class="card-subtitle">Context-Free Grammar — JSON Schema 约束解码的语法基础</p>

<Band tone="tan">
  <CoreDef>JSON 属于上下文无关语言（CFG 可识别），而非正则语言（FSM 可识别）。这个理论区别有实际影响：FSM 无法原生处理递归嵌套 JSON 结构，CFG 可以。</CoreDef>
</Band>

<Band tone="paper-sunk" dense>
  <p class="advantage-note">CFG vs FSM 的实际优势：嵌套对象 <code>{"{field: {nested: ...}}"}</code> 需要下推自动机的栈记忆来追踪嵌套层级——正则语言无法表达这种递归结构。</p>
</Band>
