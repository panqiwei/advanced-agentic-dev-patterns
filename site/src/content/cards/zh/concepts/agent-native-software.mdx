<Band tone="tan">
  <CoreDef>不是先做 GUI 再适配 agent。Agent 与人类的输入/输出/状态/发现需求差异大到足以推导出不同的软件形态——agent-native 是这个推导的结果。</CoreDef>
</Band>

<Band tone="paper-raised" grow>
  <PillarGrid cols={3}>
    <Pillar label="结构化优先">所有返回可切换到 JSON；错误是带 code + remediation 的对象，不是 traceback</Pillar>
    <Pillar label="自描述">--help / schema / metadata 回答"你能做什么"；文档与代码同一工具</Pillar>
    <Pillar label="可组合">每个命令做一件事，可管道；状态显式可序列化为文本</Pillar>
    <Pillar label="显式状态">长任务可 save/load；支持 undo 原语；不依赖连续会话</Pillar>
    <Pillar label="幂等性">副作用标记清晰；只读 vs 写 vs 不可逆；失败可重试</Pillar>
    <Pillar label="POSIX 优先">复用 agent 已掌握的工具：shell / pip / env var；避免自研 SDK</Pillar>
  </PillarGrid>
</Band>
