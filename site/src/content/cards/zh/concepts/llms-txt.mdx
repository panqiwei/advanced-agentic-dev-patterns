  <h2 class="card-title">llms.txt</h2>
  <p class="card-subtitle">robots.txt 告诉爬虫可以爬什么；llms.txt 告诉 LLM 网站是什么</p>

<Band tone="tan">
  <CoreDef>放置于网站 /llms.txt 的 Markdown 文件——主要用于推理时（用户向 LLM 询问时），而非训练时。网站主动提供 LLM 可解析格式，而非等待 LLM 自行抓取解析 HTML。</CoreDef>
</Band>

<Band tone="paper-raised" grow>
  <div class="format-spec">
    <Kicker>格式规范</Kicker>
    <div class="fs-row"><span class="fs-req">必填</span><span class="fs-el">H1</span><span class="fs-desc">项目或站点名称</span></div>
    <div class="fs-row"><span class="fs-opt">可选</span><span class="fs-el">Blockquote</span><span class="fs-desc">一句话摘要（理解其余内容所必需）</span></div>
    <div class="fs-row"><span class="fs-opt">可选</span><span class="fs-el">Markdown 段落</span><span class="fs-desc">补充背景，不含标题</span></div>
    <div class="fs-row"><span class="fs-opt">可选</span><span class="fs-el">H2 分节</span><span class="fs-desc">带说明的超链接列表</span></div>
    <div class="fs-row special"><span class="fs-spec">特殊</span><span class="fs-el">Optional 节</span><span class="fs-desc">context 紧张时可跳过，放次要资源</span></div>
  </div>
  <div class="compare-table">
    <Kicker>与现有标准的关系</Kicker>
    <div class="ct-row ct-header"><span>标准</span><span>面向对象</span><span>使用时机</span></div>
    <div class="ct-row"><span>robots.txt</span><span>搜索爬虫</span><span>索引时</span></div>
    <div class="ct-row"><span>sitemap.xml</span><span>搜索引擎</span><span>索引时</span></div>
    <div class="ct-row highlight"><span>llms.txt</span><span>LLM</span><span>推理时（用户请求时）</span></div>
  </div>
  <KV label="配套约定">URL + .md 提供 Markdown 版内容，让 LLM 获取干净文档无需解析 HTML</KV>
</Band>
