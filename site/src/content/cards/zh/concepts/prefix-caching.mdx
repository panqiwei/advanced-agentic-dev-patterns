<Band tone="tan">
  <CoreDef>Agent loop 的 prefill/decode 比约 100:1，前缀缓存命中率直接决定成本。以 Claude Sonnet 为例：未缓存 $3.00/MTok，缓存命中 $0.30/MTok——10× 差距。从 0 到 80% 命中率可使每任务 API 成本降低约 8×。</CoreDef>
</Band>
