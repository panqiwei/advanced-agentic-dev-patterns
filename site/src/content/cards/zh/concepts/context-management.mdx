<Band tone="tan">
  <CoreDef>LLM 患有"顺行性遗忘"——weights 固定、context window 每个 session 清零。人类同事会积累上下文、巩固记忆、发展专业知识；LLM 不会原生做到这些，context 是工作记忆，必须被显式编程。</CoreDef>
</Band>

<Band tone="paper-raised" grow>
  <div class="management-layers">
    <div class="ml-section">
      <Kicker>Context Window 内</Kicker>
      <KV label="Compaction">Claude Agent SDK 内建：context 接近上限时压缩对话历史</KV>
      <KV label="工具结果管理">选择性保留或丢弃工具调用结果</KV>
    </div>
    <div class="ml-section">
      <Kicker>跨 Session（外部化状态）</Kicker>
      <KV label="Progress file">claude-progress.txt：每 session 末写入结构化进度摘要</KV>
      <KV label="Git history">commit message 作为可审计的变更记录</KV>
      <KV label="Feature list">任务完成度的客观指标，防止 session 间信息断层</KV>
    </div>
  </div>
</Band>
