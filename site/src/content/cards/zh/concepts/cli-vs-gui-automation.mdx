<Band tone="tan">
  <CoreDef>不是战术差异，而是对"agent 与软件应该如何相处"的根本哲学分歧。GUI 让 agent 像人一样操作 UI；CLI 给软件套结构化接口让 agent 直接调用功能。</CoreDef>
</Band>

<Band tone="paper-raised" grow>
  <div class="compare-table">
    <div class="ct-header">
      <span>维度</span><span class="col-gui">GUI Automation</span><span class="col-cli">CLI Generation</span>
    </div>
    <div class="ct-row"><span>接口</span><span>像素 / Accessibility Tree</span><span>命令字符串 / JSON</span></div>
    <div class="ct-row"><span>生成时成本</span><span class="cost-low">低（无需分析内部）</span><span class="cost-high">高（需生成/测试/维护）</span></div>
    <div class="ct-row"><span>运行时成本</span><span class="cost-high">高（截图 tokens + 慢速）</span><span class="cost-low">低（纯文本，毫秒响应）</span></div>
    <div class="ct-row"><span>稳定性</span><span class="cost-high">低（UI 改版即崩）</span><span class="cost-low">高（CLI 命令相对稳定）</span></div>
    <div class="ct-row"><span>可并行性</span><span class="cost-high">难（独占屏幕）</span><span class="cost-low">天然并发</span></div>
    <div class="ct-row"><span>可审计</span><span class="cost-high">弱（截图记录）</span><span class="cost-low">强（命令历史即 log）</span></div>
  </div>
</Band>

<Band tone="paper-sunk" dense>

    </div>
  </div>
</Band>
