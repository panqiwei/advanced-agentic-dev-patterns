  <p class="card-subtitle">Claude Code Permission System — 基于可逆性的分层工具审批</p>

<Band tone="tan">
  <CoreDef>三级结构背后的设计逻辑是可逆性：只读操作无副作用可自由执行；文件修改可撤销（git），会话级授权够用；Bash 命令可能执行任意系统操作，需要永久记录。</CoreDef>
</Band>

<Band tone="paper-raised" grow>
  <div class="tier-stack">
    <div class="tier t-read">
      <span class="tier-name">只读工具</span>
      <span class="tier-examples">Read · Grep · Glob</span>
      <span class="tier-policy">无需审批</span>
    </div>
    <div class="tier t-write">
      <span class="tier-name">文件修改工具</span>
      <span class="tier-examples">Edit · Write</span>
      <span class="tier-policy">"不再询问" → 会话内永久授权</span>
    </div>
    <div class="tier t-bash">
      <span class="tier-name">Bash 命令</span>
      <span class="tier-examples">Shell 执行（任意系统操作）</span>
      <span class="tier-policy">"不再询问" → 保存为永久 allow 规则</span>
    </div>
  </div>
  <div class="rule-syntax">
    <Kicker>规则语法示例</Kicker>
    <div class="rule-examples">
      <code>allow: ["Bash(npm run *)", "WebFetch(domain:github.com)"]</code>
      <code>deny: ["Bash(git push *)", "Read(~/.ssh/**)"]</code>
    </div>
  </div>
</Band>

<Band tone="paper-sunk" dense>

  </div>
</Band>
