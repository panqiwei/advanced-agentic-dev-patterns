<Band tone="paper-raised" dense>
  <h2 class="card-title">干预主义理论</h2>
  <p class="card-subtitle">Interventionist Theory — X 对 Y 有因果影响 iff 存在对 X 的干预能改变 Y</p>
</Band>

<Band tone="tan">
  <CoreDef>因果不是静态关系，而是"如果你操控 X 会怎样"的系统性回答。与 Lewis 反事实理论的区别：反事实通过最近可能世界评估；干预主义通过固定因果无关因素后的实际操控评估。</CoreDef>
</Band>

<Band tone="paper-raised" grow>
  <div class="woodward-defs">
    <Kicker>Woodward 定义链（非恶性循环）</Kicker>
    <div class="def-row">
      <span class="def-n">1</span>
      <div>
        <span class="def-title">干预变量 I</span>
        <span class="def-body">I 影响 X；I 使 X 不受其他变量控制；I→Y 的所有路径经过 X；I 与 Y 的非 X 路径影响因素独立</span>
      </div>
    </div>
    <div class="def-row">
      <span class="def-n">2</span>
      <div>
        <span class="def-title">直接影响</span>
        <span class="def-body">存在对 X 的干预，固定其他变量后改变 Y</span>
      </div>
    </div>
  </div>
  <div class="sem-tools">
    <Kicker>结构方程形式化</Kicker>
    <KV label="非对称结构方程">D := C 表达 C 对 D 的因果方向，不同于对称的相关关系</KV>
    <KV label="模块性">每个方程可独立被干预破坏，不影响其他方程</KV>
    <KV label="A/B 测试对应">固定其他变量，改变一个变量——这就是干预的工程实现</KV>
  </div>
  <div class="two-paths">
    <span class="tp-label">因果优先 vs 反事实优先</span>
    <span class="tp-text">Pearl/Woodward：结构方程表示因果依赖，由此推导反事实 | Lewis 路线：结构方程表示反事实依赖</span>
  </div>
</Band>
