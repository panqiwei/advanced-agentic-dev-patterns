# 正交

Agentic system 的输出是两股力的合成：模型能力，加上你在 harness 层做的一切。模型那股力你控制不了，但可以观察、可以预判。它在持续增强，底层架构甚至可能发生根本性的变化。

在这股力面前，harness 工程师该做的不是追着它跑，而是找到与它**正交**的方向——做模型变强也替代不了的事，让每一份投入都在累积而不是在消耗。
